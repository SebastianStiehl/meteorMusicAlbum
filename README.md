# meteorMusicAlbum - coding dojo II



## setup

* install meteor:
```
curl https://install.meteor.com | sh
```

* download project sources
```
git clone https://github.com/5oc/meteorMusicAlbum.git
```

* start app
```
cd ./meteorMusicAlbum
meteor
```

* reset database
```
meteor reset
```
